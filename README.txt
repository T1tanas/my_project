 This application is used to scrape todays name days from 'www.days.lt' 
That scrapped data (names) can be added to your WP page.

***** INSTRUCTIONS*****
** Add this  *.zip file as a custom plugin to your WP site and activate it ***
** To add data to page you would need to use shortcode function.**
** Just add shortcode snipet via page edit to any desired place inside a webpage***
*** and insert tag [names] to display data in list form.***
